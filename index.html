Sorteador de números

  • Gera uma lista de números inteiros de acordo com os limites que você definir nos campos De e Até e os sorteia aleatoriamente
  • Esta página possui autovalidação, porém, certifique-se de preencher os campos corretamente
  • Não é possível sortear o mesmo número mais de uma vez
  • Não é possível sortear mais um se o número de sorteados já foi atingido
  • Para realizar um novo sorteio basta clicar no botão RESETAR, no final da lista de sorteados
